# PR Doctor 🩺 — Agentic DevOps PR Reviewer + Auto-Fix

PR Doctor is an agentic DevOps assistant that analyzes GitHub Pull Requests for high-risk issues (secrets, debug logs, SQL injection patterns, missing tests), generates a safe patch, posts a structured review comment, and can open an auto-fix PR.

## Live Demo Proof (GitHub)
- Demo repo: https://github.com/M10vir/pr-doctor-demo-repo
- Bad PR: https://github.com/M10vir/pr-doctor-demo-repo/pull/1
- Auto-fix PR (generated by PR Doctor): https://github.com/M10vir/pr-doctor-demo-repo/pull/4

## What it does
1) Create a Run (stores PR URL + status)
2) Analyze PR (heuristics + file/line hints)
3) Generate Patch (safe unified diff)
4) Comment Review on PR (agent action proof)
5) Open Fix PR (branch + commit + PR)

## Tech Stack
- Backend: FastAPI + GitHub REST API + SQLAlchemy (runs history)
- Frontend: React + Vite dashboard
- Deploy: Azure Container Apps (API). Frontend deploy to Azure Static Web Apps (recommended)

## API Endpoints
- `GET /health`
- `GET /runs`, `POST /runs`, `GET /runs/{id}`
- `POST /analyze-pr`
- `POST /generate-patch`
- `POST /comment-review`
- `POST /open-fix-pr`
